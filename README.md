# VIBE — Supplementary Materials

Open `index.html` in a modern browser. No build step required.

## Layout

```
samples/sample_{1,2}/         per-model video outputs for comparison
instruction_following/video_1/ source video + 4 prompt variants
rhythm/                        dance + rhythm-aligned music
gallery/v2m/                   video-to-music gallery
gallery/t2m/                   text-to-music gallery (audio + waveform image)
```

Replace placeholder media files in those folders to populate the page.
